# Usage

<b>Start interactive session</b>

```bash
kana
```

<br/>
<b>Switch decks</b>

```bash
kana deck
```
<br/>
<b>Add a card</b>

```bash
kana add
```

or

```bash
kana card add -f "front" -b "back"
```

<br/>
<b>Study cards</b>

```bash
kana study
```
